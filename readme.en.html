Overview

Video Codec List is a Windows tool to enumerate and list video codec. It can list VCM codecs, DMO codecs and DirectShow native codecs. In x64 Windows, it can also list x64 version codecs. (Note that DirectShow decoder is not enumerated, because "decoder category" does not exist in DirectShow.)

Environment

Minimum requirement

OS
Windows XP or later

Installation

Expand all files to suitable folder and execute vclist.exe. This application doesn't write to filesystem nor registry, so you can uninstall it just by deleting expanded files.
