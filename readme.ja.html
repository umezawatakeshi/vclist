概要

Video Codec List は、映像コーデックを列挙して一覧するための Windows 用のツールです。VCM コーデック、DMO コーデック、DirectShow ネイティブエンコーダ および Media Foundation コーデックを一覧できます。x64 Windows においては、x64 版のコーデックも一覧できます。なお、DirectShow においては「デコーダ」というカテゴリは存在しないので、列挙しません（空の表示になります）。

動作環境

必須環境

OS
Windows XP 以降

インストール等

適当なフォルダに全てのファイルを展開し、vclist.exe を実行してください。アプリケーション自体はファイルシステムやレジストリに書き込まないので、展開したファイルを削除すればアンインストール完了です。
